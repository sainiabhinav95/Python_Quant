## Installation:
#### Virtual Environment Method:  
Create a virtual environment anywhere: 

>python -m venv .venv

Activate it:  

>source .venv/bin/activate

Install the package:

>pip install python_quant    

Start the Web App:
>start_app

#### System-wide Installation:
Directly install using pip:
> pip install python_quant
